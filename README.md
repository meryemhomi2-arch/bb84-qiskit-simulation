# bb84-qiskit-simulation
A basic implementation of BB84 quantum key distribution protocol using Qiskit and Python
# BB84 Qiskit Simulation

# Description
This project simulates the BB84 quantum key distribution protocol using Qiskit and Python.

# Tools
- Python
- NumPy
- Qiskit

# What it does
- Basis selection: Alice and Bob choose random bases for encoding/decoding.
- Eavesdropper simulation: Eve tries to intercept the key.
- QBER calculation: Measures the error rate in the shared key.

# Goal
To understand basic quantum key distribution and practice Python programming with Qiskit.
